WaveSync Documentation

Welcome to the WaveSync documentation! Here you'll find all the information you need to get started and make the most of our music experience.

Getting Started

To start using WaveSync, simply invite the bot to your server and use the commands to control the music. Type `+help` to get a list of available commands.

Features

WaveSync offers various features like music playback, playlists, volume control, and more! Explore the bot commands to enhance your server's experience.
